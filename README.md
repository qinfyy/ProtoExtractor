# ProtoExtractor

该工具用于将 protoc 编译的代码转换为 protobuf 定义文件。

## 环境要求
- [Python 3.13](https://www.python.org/)

## 支持的语言
- CSharp
- Java
- Python
- Ruby
- Php
- C++
- Go
- Rust Prost
- zig-protobuf
- python-betterproto
- protobuf-net
- protobuf-net (VB.NET)

## 用法
```
pip install -r requirements.txt
python main.py <args>
```
